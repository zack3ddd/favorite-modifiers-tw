# Favorite Modifiers（繁體中文延伸版）

在 Blender 的修改器面板上方，顯示一排「常用修改器」快速按鈕，點一下就加入。繁體中文化，並延伸多項功能。

> 原始創意與基礎程式碼來自 **Oleg Stepanov** 的 Favorite Modifiers。本版本為繁體中文化＋功能延伸版，由 Zack3D 延伸。

## 功能

- 修改器面板上方一排**常用修改器按鈕**，點擊即加入該修改器
- 常用清單**依物件類型分組**（網格 / 曲線·文字·曲面 / 晶格 / 蠟筆）
- **右鍵按鈕**：左移 / 右移 / 移除
- **偏好面板**可依物件類型直接編輯常用清單，並與介面按鈕**即時同步**
- **預設清單**（`defaults.json` 隨外掛一起發佈）＋ 儲存為預設 / 恢復預設 / 清空
- 加入修改器時**依物件類型過濾**，避免加到不適用的
- tooltip 顯示各修改器**自己的名稱與說明**，跟隨使用者語系

## 安裝

1. 下載本外掛的 zip（或到 Releases 下載）
2. Blender → 編輯 > 偏好設定 > 附加元件 > 從磁碟安裝 → 選 zip → 啟用「Favorite Modifiers」

## 使用

- 選取物件，其修改器面板上方會出現常用按鈕，點擊即加入該修改器
- 沒有任何常用時，會顯示「按 ＋ 加入常用修改器」引導
- **右鍵**常用按鈕 → 左移 / 右移 / 移除
- 偏好設定裡可依物件類型編輯清單、設定預設（儲存為預設 / 恢復預設 / 清空）
